---
description: How to deploy the Next.js application to Vercel
---

1. Log in to [Vercel](https://vercel.com) using your GitHub account.
2. Click **Add New...** -> **Project**.
3. Import the `ship-app` repository.
4. Add any necessary **Environment Variables** (e.g., `KV_REST_API_URL`, `DATABASE_URL`).
5. Click **Deploy**.
6. (Optional) Install Vercel CLI with `npm install -g vercel` and run `vercel --prod` from the terminal.
